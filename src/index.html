ברוכים הבאים לספרייה

כדי להציג את כל הספרים, ניתן להשתמש בבקשת GET לנתיב /books

כדי להציג ספר ספציפי, ניתן להשתמש בבקשת GET לנתיב /books/(id)

כדי להוסיף ספר חדש, ניתן לשלוח בקשת POST לנתיב /books
ולהוסיף ב-body את שם הספר (title) ושם המחבר (author) בפורמט JSON

כדי לעדכן ספר קיים, ניתן לשלוח בקשת PUT לנתיב /books/(id)

כדי לעדכן ערך בספר ניתן לעשות כן בבקשת PATCH

כדי למחוק ספר, ניתן לשלוח בקשת DELETE לנתיב /books/(id)
